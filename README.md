# A.E.G.I.S.: The Concrete Algorithm
A narrative-driven psychological UI simulator built for the Apogee GameJam. As the "Invisible Hand" overseeing an optimized OS, you manipulate emergency petitions and visually unmake a 16-bit RPG world into concrete blocks to achieve total, calculating efficiency.

Built using **100% Vanilla JavaScript, HTML5 Canvas API, and Web Audio API** with zero external game engines or image assets.
